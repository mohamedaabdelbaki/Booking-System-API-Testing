# Booking System API Testing

## Website
https://automationintesting.online

## Project Type
API Testing using Postman

## Test Workflow

1. Retrieve all rooms
   - Verify the third room is a suite

2. Get room details
   - Retrieve double room details
   - Verify TV feature exists

3. Book a room
   - Book a double room for 3 days
   - Provide booking details

4. Admin verification
   - Login as admin
   - Verify booking appears in report
   - Confirm booking name matches reservation

5. Add new room
   - Create a family room
   - Add three features

6. Contact support
   - Send message using contact endpoint

## Testing Techniques
- API workflow testing
- Response validation
- Authentication testing
- Admin authorization testing
- Data verification

## Tools
- Postman
- Collection Runner
